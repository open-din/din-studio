# Overview

[DIN Studio](../../README.md) is a desktop workspace for real-time audiovisual patching: visual graph editing, nodes that mirror the runtime and produce usable code, and project workflows (launcher, assets, source control, MCP).

Product intent in this repository is summarized in [`project/SUMMARY.md`](../../project/SUMMARY.md). Contributor workflow expectations live in [`project/USERFLOW.md`](../../project/USERFLOW.md). Executable scenarios and IDs are listed in [`project/TEST_MATRIX.md`](../../project/TEST_MATRIX.md).

## What this folder contains

- **Topic pages** (this section) connect product areas to numbered feature files under `project/features/*.feature.md` and to hand-written docs under `docs/*.md`.
- **Node reference** lists every palette node with a link to its feature spec: [nodes/index.md](./nodes/index.md).

Authoritative behavior is always the feature files plus the implementation; these pages are a navigable map, not a second specification.
